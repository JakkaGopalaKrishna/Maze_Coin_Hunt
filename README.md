# 🪙 Maze Coin Hunt

Maze Coin Hunt is a fun and interactive browser-based maze game where players navigate through a maze, collect coins, and try to complete the challenge with the highest possible score.

# 🎯 How to Play

1. Start the game.
2. Use keyboard arrow keys to move through the maze.
3. Collect all the coins.
4. Avoid getting stuck in the maze.
5. Complete the challenge with the maximum score.
